Google Sheets Setup

1. Open Google Sheets and create a new spreadsheet named "Nwobi Content Log"

2. Add these column headers in Row 1: Date, Day, Category, Hook, Concept, Style, Filmed

3. Go to Extensions → Apps Script

4. Paste the script from the README in your GitHub repo

5. Click Deploy → New Deployment → Web App

6. Set access to "Anyone" and copy the URL into the field above.
